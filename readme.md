# REviewer

**REviewer** is a specialized Speedrun Tool (SRT) designed for the original versions of Resident Evil games. It focuses on providing real-time data visualization, inventory tracking, and splitting capabilities, with a strong emphasis on compatibility with **Biorand** and standard releases.

> [!WARNING]
> **Alpha Release**: REviewer is currently in version `1.1.0-alpha`. Features are under active development and may change. Use with caution.

## Key Features

- **Real-time Health Monitoring**: Visual health bars and status indicators (Fine, Caution, Danger, Poison).
- **Enemy Tracker**: Live tracking of enemy HP and status in the current room.
- **Inventory Management**: View real-time inventory slots, equipped weapons, and item box contents.
- **Key Item Tracker**: Automatically tracks localized key items to ensure you never miss a critical pickup.
- **Overlay System**: Customizable in-game overlay with scaling support (1x to 3x).
- **Split Timer**: Integrated segment timer with IGT (In-Game Time) tracking.
- **Multi-Game Support**: Seamless switching between supported Resident Evil titles.

## Installation & Usage

1.  **Download**: Get the latest release from the [Releases Page](https://github.com/namsku/REviewer/releases).
2.  **Configuration**: 
    - Open REviewer.
    - Go to **Settings** and configure your game paths (e.g., save data location).
3.  **Launch**: Select your game version from the dropdown and click "Run".
4.  **Overlay**: Use the settings to position and scale the overlay to your preference.
